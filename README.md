🧱 CodeCraft Framework — Full‑Stack Open‑Source Toolkit

CodeCraft Framework est un framework open‑source complet regroupant des outils frontend, backend JavaScript, backend PHP, sécurité, SQL, et des composants UI/UX réutilisables.
Il est conçu pour accélérer la création de projets web modernes, structurés, sécurisés et cohérents.

Composants responsives prêts à l’emploi

🧩 Utilisation
Chaque module peut être utilisé indépendamment :

importer un composant UI

réutiliser un middleware Node.js

intégrer un formulaire PHP sécurisé

utiliser un modèle SQL

importer les variables SCSS

piocher dans les helpers

Le framework est pensé pour être modulaire, flexible et scalable.
